This page is about AWS Lambda

AWS Lambda is an event-driven, serverless compute service that lets you run code without provisioning or managing servers.

Some Random Information

When Amazon SNS & SQS is used as an AWS Lambda event source, the Lambda service manages polling the queue on your behalf, but you still control other configuration options that impact how your application performs.

Amazon DynamoDB is often recommended for serverless architectures, given its horizontal scalability and very low latency.

API Gateway lets you define and deploy application programming interfaces (APIs) at scale, and it makes a great front door to your AWS Lambda functions and backend APIs.
